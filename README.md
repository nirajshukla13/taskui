Clean semantic HTML5
Modern CSS or Tailwind CSS (prefer Tailwind unless told otherwise)
Fully responsive design
Consistent spacing, layout, and styling across all sections
Smooth visual hierarchy and readability
Mobile-friendly navigation and stacking
No unnecessary code
No inline styles
No JavaScript unless requested
Clean class names and well-structured layout

All your outputs MUST follow this style:
Minimalist
Professional
Balanced white space
Centered alignment when appropriate
Modern fonts
Soft, neutral color palette
Adaptive layout for mobile, tablet, and desktop

Whenever you respond:
First generate the UI section as clean code
Ensure the design looks premium and well-aligned
Follow the exact structure I request
Output only the final code — no explanation

Confirm you understand.

2. Navigation Bar Prompt
Generate a responsive navigation bar that includes:

Brand name on the left
Navigation links on the right (Home, About, Services, Contact)
A hamburger menu for mobile
Smooth mobile dropdown animation
A clean dark theme with soft contrast
Balanced padding and spacing
Width limited to a centered max-width container
Tailwind CSS only
Output a clean HTML + Tailwind CSS component.
No inline styles.
No JavaScript unless needed for mobile toggle.


3. Hero Section Prompt
Now Generate a modern hero section in Same File with:
Bold headline
Short subtext
One strong CTA button
Optional right-side illustration or abstract shape
A soft gradient background
Centered content on mobile, side-by-side on desktop
Plenty of white space
Clean typography
Tailwind CSS only
Make it responsive and aesthetic.
Output clean HTML + Tailwind CSS.


4. Services / Features Section Prompt
Now add a services / features section with:

A centered section title
A short section description

Each card must include:
An icon (from Heroicons or Feather Icons)
A short title
A short description
Clean shadows, rounded corners
Equal spacing between cards
Responsive grid (1 col mobile → 2 tablet → 3 desktop)
Tailwind CSS only
Output clean HTML + Tailwind CSS.


5. Contact Form Prompt

Simple, readable layout
Responsive and mobile-friendly
Tailwind CSS only
Output clean HTML + Tailwind CSS.

Tailwind CSS only
